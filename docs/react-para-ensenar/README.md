Esta carpeta esta pensada para explicar React basico usando este mismo proyecto como ejemplo real.

Orden recomendado:

## Objetivo

La idea no es solo definir React, sino tener herramientas para ensenarlo:

- explicaciones simples
- analogias
- mapas conceptuales
- preguntas para comprobar comprension
- ejercicios pequenos
- ejemplos reales sacados de este repo

## Sugerencia de dinamica

Primero:
- mostrar `src/Pages/App.jsx`
- explicar que React divide la UI en piezas

Despues:
- abrir `src/components/home/HomeSection.jsx`
- mostrar que un componente puede usar otros componentes

Luego:
- abrir `src/components/shared/SectionHeading.jsx`
- explicar props con un ejemplo pequeno

Finalmente:
- revisar `FavoriteCard`, `CatalogCard` y `FavoritesGrid`
- mostrar composicion y reutilizacion
